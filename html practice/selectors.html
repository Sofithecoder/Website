How do HTML selectors work?

1.Universal selector AKA *

Selector * can style the whole document. No matter what kind of elements you have in the document, * affects each of them

2.Type selectors

Type selectors are h1, h2, p and etc. they affect the whole type. if you style one P each paragraphs will be styled in the document

3. ID selectors

ID selectors style one specific element with a unique ID

4. Class selectors

Class selectors can group elements into classes and affect only those with a specific class. ID can effect only one element whereas class can affect several. aditionally, one element can only have one ID but one element can have multiple classes. in order to style a class selector you should add "." in front of the selector in style tag

5. Atribute selector

If you want to style elements depending on whether they have a specific atribute or not you should use atribute selector. If the atribute exists HTML wikl style it, if not- not.
